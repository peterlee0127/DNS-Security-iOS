# Enable DNS Security

## iPhone and iPad

1. Open DNS Security.
2. Choose a DNS profile from the Config tab.
3. Turn on DNS in the Connect tab.
4. Open iOS Settings.
5. Go to General - VPN & Network - DNS.
6. Select DNS Security.

If DNS is not active after returning to the app, open the guide again from the Connect or About screen and confirm the selected DNS profile in Settings.

## Mac

On macOS 13 or later:

1. Open System Settings.
2. Go to Network.
3. Open VPN & Filters.
4. Select Filters & Proxies.
5. Enable DNS Security.

On older macOS versions, open System Preferences - Network, choose DNS Security, and make the service active.
